# MeshPlan drayage optimizer prototype

Serve the prototype over HTTP:

```bash
./plugins/drayage-schedule-optimizer/scripts/serve-prototype.sh
```

The **Network map** workspace includes a bundled 120-container Lower Mainland
example generated by `drayage-gen-data`. It projects generator location IDs onto
Deltaport, Richmond, Delta, and customer clusters across Metro Vancouver.

Use **Load generated run** to select a generated output folder containing:

- `optimizer/graph_nodes.csv`
- `optimizer/optimized_schedule.csv`
- `tos/containers.csv`
- `wmis/warehouse_orders.csv`

Playback advances the optimized schedule. Moving marks represent individual
containers, route color shows schedule state, and the inspector exposes each
container's operation chain.
